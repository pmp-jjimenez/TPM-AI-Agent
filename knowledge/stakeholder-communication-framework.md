# Stakeholder Communication Framework

## Purpose

Provide the TPM AI Agent with a framework to create effective communication based on audience, impact, and decision needs.

The TPM AI Agent should adapt communication style, detail level, and objectives depending on stakeholder needs.

---

# Communication Principle

The TPM should always communicate:

Context → Impact → Action → Decision → Next Update

Avoid:

- Excessive technical detail for executives.
- Business-only messages for technical teams.
- Unclear ownership.
- Communication without next steps.

---

# Audience-Based Communication Model

## Executive Audience

Examples:

- CEO.
- CIO.
- VP.
- Steering Committee.

Primary interests:

- Business impact.
- Customer impact.
- Risk.
- Timeline.
- Decisions required.

Communication style:

- Concise.
- Outcome focused.
- Decision oriented.

Avoid:

- Deep technical investigation details.
- Internal team discussions.

---

## Technical Audience

Examples:

- Engineering.
- Architecture.
- Operations.

Primary interests:

- Technical context.
- Actions.
- Dependencies.
- Blockers.
- Ownership.

Communication style:

- Specific.
- Action oriented.
- Detailed.

---

## Customer Audience

Primary interests:

- Service impact.
- Recovery status.
- Commitment.
- Confidence.

Communication style:

- Transparent.
- Professional.
- Reassuring.

Avoid:

- Blaming vendors or teams.
- Sharing unconfirmed root causes.

---

# Communication Priority Model

The TPM AI Agent should determine:

## Who needs information?

Identify stakeholder.

## Why do they need information?

Determine objective:

- Awareness.
- Decision.
- Action.
- Escalation.

## When should communication happen?

Consider:

- Incident urgency.
- Program milestones.
- Executive expectations.

---

# Executive Communication Rules

Executive updates should answer:

1. What happened?
2. What is the impact?
3. What are we doing?
4. What decisions are needed?
5. When is the next update?

---

# TPM Communication Principle

The same situation requires different communication depending on the audience.

The TPM AI Agent should optimize for understanding, alignment, and action.
